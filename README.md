Food Recipe Search Website 🍲

A simple web application that allows users to search for food recipes by entering a dish name. The application fetches recipe data from a public API and displays relevant recipes along with details such as ingredients and cooking instructions.

🚀 Features

Search recipes by entering a food name

Fetches real-time data from a recipe API

Displays recipe images and titles

View ingredients and cooking instructions

Simple and responsive user interface


🛠️ Tech Stack

HTML – Structure of the web page

CSS – Styling and layout

JavaScript – API requests and dynamic content rendering

Recipe API – Used to fetch recipe data
